## 作用域

与python不同, 通过**花括号**就可以让变量只作用于作用域内
```java
{
    ...
    int i = 0; // 变量i从这里开始定义
    ...
    {
        ...
        int x = 1; // 变量x从这里开始定义
        ...
        {
            ...
            String s = "hello"; // 变量s从这里开始定义
            ...
        } // 变量s作用域到此结束
        ...
        // 注意，这是一个新的变量s，它和上面的变量同名，
        // 但是因为作用域不同，它们是两个不同的变量:
        String s = "hi";
        ...
    } // 变量x和s作用域到此结束
    ...
} // 变量i作用域到此结束
```

## 运算规则

和其他语言类似, 相比于python需注意**溢出**问题
与 &
或 |
非 ~
异或 ^
无符号右移`>>>`, 不管怎么样最高位为零, 所以会变成负数
对`byte`和`short`类型进行移位时，会首先转换为`int`再进行位移。

仔细观察可发现，左移实际上就是不断地×2，右移实际上就是不断地÷2。

### 位运算

在Java的计算表达式中，运算优先级从高到低依次是：

- `()`
- `!` `~` `++` `--`
- `*` `/` `%`
- `+` `-`
- `<<` `>>` `>>>`
- `&`
- `|`
- `+=` `-=` `*=` `/=`

## 强制转型

例如`int`和`short`进行运算的时候, 会自动转换成`int`, 因为`int`比`short`大
也可以强制转型
e.g.
```java
int i = 12345;
short s = (short) i; // 12345
```
括号内就是转型目标, 但**不可以越界**, 否则溢出导致高位丢失

### 类型提升

和`short`提升到`int`类似, 当运算中出现浮点数时, 结果会直接升级到浮点
	tips
	如果出现复杂四则运算例如![[Pasted image 20250827104608.png]]
	因为优先计算24/5这样结果会是整数,没法自动转变
	所以要把其变成`1.2+24.0/5`

也可以强制转型, 例如
```java
int n1 = (int) 12.3; // 12
int n2 = (int) 12.7; // 12
int n3 = (int) -12.7; // -12
int n4 = (int) (12.7 + 0.5); // 13
int n5 = (int) 1.2e20; // 2147483647
```
会直接**丢弃小数点后的数字**, 若想四舍五入**加上0.5**就行
