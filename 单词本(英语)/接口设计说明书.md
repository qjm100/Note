# 智能教务管理系统 - 接口设计说明书

## 1. 概述

### 1.1 项目简介
智能教务管理系统是一个基于Spring Boot + MyBatis + Shiro技术栈的教育管理平台，提供学生管理、教师管理、课程管理、班级管理、题库管理、试卷管理、成绩管理、考试记录等核心功能。

### 1.2 技术架构
- **后端框架**: Spring Boot 2.5.15
- **安全框架**: Apache Shiro 1.13.0
- **持久层**: MyBatis + PageHelper
- **数据库连接池**: Druid 1.2.23
- **API文档**: Swagger 3.0.0
- **Java版本**: JDK 1.8

### 1.3 接口规范

#### 1.3.1 通用响应格式
所有接口返回统一的JSON格式：

```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {}
}
```

#### 1.3.2 状态码说明
| 状态码 | 说明 |
|--------|------|
| 200 | 操作成功 |
| 500 | 操作失败 |
| 401 | 未授权 |
| 403 | 禁止访问 |

#### 1.3.3 分页参数
| 参数名      | 类型      | 必填  | 说明        |
| -------- | ------- | --- | --------- |
| pageNum  | Integer | 否   | 页码，默认1    |
| pageSize | Integer | 否   | 每页条数，默认10 |

#### 1.3.4 分页响应格式
```json
{
  "total": 100,
  "rows": [],
  "code": 200,
  "msg": "查询成功"
}
```

## 2. 学生管理模块

**接口地址**: `POST /edu/student/list`
**权限要求**: `edu:student:list`

**请求参数**:

|---------|------|------|------|

## 11. 安全说明

### 11.1 认证方式
系统采用Shiro进行权限认证，所有接口需要在请求头中携带Token：
```
Authorization: Bearer {token}
```

### 11.2 权限控制
每个接口都配置了相应的权限标识，只有拥有对应权限的用户才能访问。

### 11.3 数据加密
- 用户密码采用Shiro加密存储
- 敏感数据传输建议使用HTTPS协议
